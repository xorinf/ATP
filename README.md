# ATP (Assignments & Tasks Portfolio)

This repository serves as a single central storage for all weekly coding assignments, exercises, and projects.

## Directory Structure

All weekly folders are structured using lowercase names (`week1`, `week2`, ..., `week7`), and all assignment filenames from Week 1 to Week 6 use `camelCase` naming conventions.

```
ATP
├── week1/                  # Basic JavaScript & Array Operations
├── week2/                  # Advanced Array Iteration, ES Modules, OOP & Express REST CRUD Basics
├── week3/                  # Express Router architecture, Middlewares, MongoDB/Mongoose & User Auth (JWT/Cookies)
├── week4/                  # Advanced Express REST APIs (Article model) & Base HTML/CSS styling
├── week5/                  # CSS layout systems (Flexbox, CSS Grid), Utility Frameworks (Tailwind) & Intro to React
├── week6/                  # React State Management, forms, Context API & Full-stack MERN integration
└── week7/                  # Production MERN stack Blog app (with Cloudinary uploads & Author/User portals)
```

Each weekly directory includes a detailed `README.md` listing the specific tasks and files covered in that week.
