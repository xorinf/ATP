# WEEK 2 Assignments

This folder contains assignments and practice tasks from Week 2, organized by date.

## 25-02-2026 - Array Methods & Modules
- `01-array-methods-1.js`: Practice with `filter`, `map`, `reduce`, `find`, and `findIndex` on simple arrays.
- `02-array-methods-2.js`: More advanced practice with array methods on arrays of objects (carts, students, employees, movies, transactions).
- `03-ecommerce-modules/`: A modular E-commerce script exploring ES Modules.
- `04-task-modules/`: A modular ToDo application structure using ES Modules.

## 26-02-2026 - Classes & Object Copying
- `01-class-book.js`: Object-Oriented Programming practice with a `Book` class.
- `02-spread-array.js`: Copying and extending arrays using the spread operator.
- `03-spread-object.js`: Copying and extending objects using the spread operator.
- `04-shallow-copy.js`: Demonstrating shallow copying and its mutation effects.
- `05-deep-copy.js`: Demonstrating deep copying using `structuredClone`.

## 28-02-2026 - Express Backend Basics
- `server.js`: An Express.js backend server with basic CRUD APIs for users and products.
- `api-requests.http`: HTTP file for testing the endpoints in `server.js`.
- `package.json` / `package-lock.json`: Dependencies for the Node application.
