# Week 5: HTML Layouts, CSS Layout (Flexbox & Grid), Tailwind CSS & Intro to React

This folder contains assignments exploring modern styling, advanced CSS layout systems, responsive design, and transition to single-page application development with React.

## Projects & Topics

### 1. Basic HTML/CSS Layouts
- `htmlLayout/`: Page layout skeleton.
- `citiesPage/`: A gallery site featuring Bangalore, Chennai, and Hyderabad.
- `cssAssignment1/` & `cssAssignment2/`: Assignments practicing borders, backgrounds, typography, and styling selectors.
- `htmlCssProject/`: General markup styling assignment.

### 2. Flexbox Layouts
- `flexboxPart1/`: Basic Flexbox alignment properties (justify-content, align-items).
- `flexboxPart2/`: Layout construction for a social profile card with links.

### 3. Grid & Responsive Design
- `gridAssignment1/`: Basic CSS Grid layouts.
- `cssGridLayout/`: Multi-column layouts and media-query breakpoints.
- `responsiveLayout/`: Desktop-to-mobile view responsive site structure.

### 4. Utility-First Frameworks
- `tcssProject/`: Custom utility classes.
- `tailwindProject/`: Introduction to Tailwind CSS utility flags and configuration settings.

### 5. React Introduction
- `reactIntro/`: First React SPA exploring Vite setup, components (Navbar, UserList, Address, Counter, Button, Footer), CSS modules, and custom data bindings.
