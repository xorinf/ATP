# Week 1: Basic JavaScript Concepts & Array Operations

This folder contains assignments and practice tasks from Week 1, focusing on foundational JavaScript concepts, functions, and array operations.

## Assignments

### 1. Basic JavaScript Concepts
- `largestOfTwo.js`: Finds the largest of two numbers using `if-else`.
- `largestOfThree.js`: Finds the largest of three numbers.
- `arraySum.js`: Calculates the sum of all elements in an array using a `for` loop.
- `arrayMin.js`: Finds the smallest element in an array.

### 2. Functional Programming in JavaScript
- `functionLargestOfThree.js`: A function that takes three arguments and returns the largest.
- `functionArraySum.js`: A function that calculates and returns the sum of elements in an array.
- `functionLinearSearch.js`: A function that performs linear search on an array and returns the element's index, or `-1` if not found.

### 3. Object Array Operations
- `employeeArrayOperations.js`: Performs insertion, deletion, and updates on an array of employee objects using array methods like `splice`.
