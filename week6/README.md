# Week 6: State Management, Context API, MERN Integration

This week deals with managing state in React, passing context to avoid prop drilling, and building full-stack MERN (MongoDB, Express, React, Node) applications.

## Projects

### 1. State Management Basics
- `userCountApp/`: A React app demonstrating stateful data tracking and event handlers to count/track users.
- `userFormTable/`: Forms in React, gathering input parameters, updating list states, and rendering data in dynamic tables.

### 2. Full-Stack MERN Integration
- `blogPrefinal/`: High-fidelity layout modeling a MERN Blog:
  - `backend/`: Express, MongoDB connections, and endpoints to manage employee datasets (`EmployeeModel`).
  - `frontend/`: Vite-based React SPA for employee CRUD operations (Header, Footer, CreateEmployee, EditEmployee, ListOfEmployee, RootLayout).

### 3. Context API
- `counterContext/`: An application demonstrating how to share state across non-hierarchical components using Context API (`ContextProvider`) and custom store wrappers.
